# ChromaCatch iOS Controller App

This package contains the **video + input controller** iOS app:

- Project: `ChromaCatchController/ChromaCatchController.xcodeproj`
- Focus: ReplayKit broadcast, backend control WebSocket, ESP32 relay, BLE HID commander
- UI tabs: Dashboard, Video, Input, Settings

Location spoofing has been split into a dedicated package at:

- `services/ios-location-spoofer/`
